# 💼 Web Career Portal

A full-stack job portal connecting **students** with **recruiters**. Students can browse and apply to jobs, while recruiters can post and manage listings.

> Built with **React**, **Node.js**, **Express**, and **MongoDB**

---

## 📌 Features

👨‍🎓 **Students**  
- Register/Login  
- View & filter jobs  
- Apply for jobs  
- Manage profile  

🧑‍💼 **Recruiters**  
- Post job listings  
- View/edit/delete jobs  
- See list of applicants  

---

## 📂 Project Structure
Web_career_portal/
├── backend/ # Express + MongoDB API
├── frontend/ # React + Tailwind UI


---

## ⚙️ Tech Stack

- **Frontend**: React, React Router, Axios, Tailwind CSS  
- **Backend**: Node.js, Express, MongoDB, JWT  
- **Tools**: Git, Vercel, Render, Postman
